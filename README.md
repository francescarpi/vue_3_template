# Vue 3 Template

This is a template to help get you started developing with Vue3, TypeScript and other packages, using Vite.

Packages included:

- Vue3
- Tailwind
- Pinia

## Usage

- Install packages

```
npm install
```

- Run dev server

```
npm run dev
```
