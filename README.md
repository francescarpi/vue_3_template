# Vue 3 Template

This is a template to help get you started developing with Vue3, TypeScript and other packages, using Vite.

Packages included:

- Tailwind
- Pinia
- VueRouter
- FormKit

## Usage

- Install packages

```
npm install
```

- Run dev server

```
npm run dev
```

## Editor

Recommended extensions for `vscode`:

- (Install): Volar
- (Install): Prettier
- (Uninstall): Vetur
